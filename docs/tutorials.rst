
Tutorials
=========

The ``/docs/tutorials`` directory contains a collection of
`Jupyter Notebooks <https://jupyter.org/>`__  walking through specific tutorials.

Launch ``jupyter`` or ``jupyter lab`` to run these notebooks interactively.

.. code:: bash

   $ cd docs/examples
   $ jupyter lab
